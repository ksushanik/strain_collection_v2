# Methods (Методы исследования/хранения)

- **Назначение**: Справочник методов, используемых для исследования или хранения штаммов.
- **Модель**: `Method` (id, title, description?).
- **API**:
  - `GET /methods` — список с пагинацией/поиском.
  - `GET /methods/:id` — детальная информация.
  - `POST /methods` — создание.
  - `PUT /methods/:id` — редактирование.
  - `DELETE /methods/:id` — удаление.
- **Фронтенд**:
  - `/methods` — список, поиск, создание, редактирование.
  - Использование в формах штаммов/образцов (если применимо).
- **Права**:
  - ADMIN/MANAGER: полный доступ (CRUD).
  - USER: только чтение.
