"# Frontend (Next.js)\n\n- **Старт**: `cd frontend && npm install && npm run dev -- -p 3001`.\n- **Навигация**: App Router, MainLayout центрирует контент (max-w-6xl), Sidebar строится по ui-bindings + статические ссылки (Media, Legend, Settings).\n- **Страницы**:\n  - `/` Dashboard — использует analytics overview.\n  - `/dynamic/[slug]` — секции из ui-bindings (strain list, sample list, storage view).\n  - `/strains/[id]` — карточка штамма, блоки taxonomy/growth/genetics/info/storage/media.\n  - `/strains/new`, `/strains/[id]/edit` — формы (returnTo для навигации).\n  - `/samples` — список/карточки проб.\n  - `/dynamic/storage` — StorageView: выбор бокса, сетка ячеек, allocate/unallocate, подсветка по query `boxId`/`cell`.\n  - `/media` — CRUD справочника питательных сред (поиск/пагинация, add/edit/delete).\n  - `/legend` — просмотр/редактирование текстовой легенды.\n  - `/settings` — управление ui-bindings (порядок, метки, иконки, routeSlug, field packs).\n- **Ключевые компоненты**: `components/domain/strain-list`, `sample-list`, `storage-view` (подсветка ячейки из query), `photo-upload`.\n- **API клиент**: `src/services/api.ts` — включает auth headers из localStorage, методы для strains/samples/storage/media/legend/ui-bindings/analytics.\n- **ReturnTo**: ссылки на штаммы/формы передают `?returnTo=...` для возврата.\n- **UI**: shadcn/ui, Tailwind; Badge/Select/Table/Card/Button.\n"***
